# 项目说明

## DM_EFT
当以暗物质的自旋分类（$0, 1/2, 1$），又可以介子味道分类（$B\to \pi, B\to K, K\to \pi$），其后又以介子电荷分类。

### Spin-0 DM
轴子只是一种赝标属性的暗物质候选者。

所涉及的过程有（仅考虑一个新标量粒子）：
- $K_L\to\pi^0(\rho) \phi$, $K_L\to\pi^+\pi^- \phi$, $\Gamma(K_L\to \gamma\phi)=0$
- $K^+\to \pi^+\phi$

- $B\to H^{(*)}\phi$,  
  其中：$H \in\{K,\pi\}$。此一处共有8个过程，包括$b\to s|d$两类，$B\to P|V$两类，荷电与中性两类。

- $K_L\to \bar{\phi}\phi$, $K_L\to \pi^0\bar{\phi}\phi$  
  此处末态中包含两个新标量

### Spin-1/2 DM
- $B\to \bar{\psi}\psi$
- $B\to H^{(*)}\bar{\psi}\psi$
